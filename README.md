# CodeSpectra

Academic Code Plagiarism Detection System

## Features
- Type 1-4 Clone Detection
- Clone-Resilient Detection (CRD)
- Support for C++ (primary)

## Tech Stack
- Frontend: React + TypeScript + Vite
- Backend: Node.js + Express
- Analysis: Python + FastAPI
- Database: PostgreSQL (to be added)

## Running the Project

### Backend
```bash
cd backend
npm run dev
```

### Analysis Engine
```bash
cd analysis-engine
venv\Scripts\activate
uvicorn main:app --reload --port 8000
```

### Frontend
```bash
cd frontend
npm run dev
```
